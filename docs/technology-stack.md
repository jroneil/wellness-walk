# Required Technology Stack

This project is NOT a greenfield project.

Existing Backend
- Spring Boot 4.1
- Java 21
- Maven
- REST APIs

Existing Frontend
- Next.js 16
- TypeScript
- React
- Tailwind CSS

Database
- None active
- PostgreSQL can be added later when a real persistence requirement exists

Requirements

- Reuse the existing Spring Boot project.
- Do NOT create a Python backend.
- Do NOT use Flask.
- Do NOT use FastAPI.
- Do NOT use Django.
- Do NOT replace the backend technology.
- All backend code must be Java.
- Follow the existing package structure.
- Do NOT add a database service or placeholder credentials without an active persistence requirement.
