# Project Rules

These rules are mandatory.

## Existing Project

This is NOT a new project.

Reuse the existing codebase.

## Backend

- Spring Boot 4.1
- Java 21
- Maven

Never use:
- Python
- Flask
- FastAPI
- Django
- Node backend

## Frontend

- Next.js 16
- TypeScript
- Tailwind CSS

## Development Rules

- Read the PRD before coding.
- Read the implementation plan before coding.
- Implement one phase at a time.
- Never skip ahead.
- Never introduce dummy data unless requested.
- Verify the project builds.
- Explain all file changes.
- Stop after the requested phase.
